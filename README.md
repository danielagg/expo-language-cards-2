# NL Language Cards

A flashcard game for learning the most common Dutch words. Built with
**Bun + Vite + React + TypeScript**, **TanStack Router**, **Tailwind v4**,
and **shadcn/ui** components. Scores persist in the browser via
`localStorage`.

## Play

- A Dutch word appears on a card. Type the English translation, or switch
  to **Multi-select** to pick from four options.
- Toggle the direction at any time: **NL → EN** (default) or **EN → NL**.
- Per-word accuracy ("you got this word 5 out of 7 times correct") shows
  under each card. Session streak and accuracy are tracked too.
- See **/stats** for overall accuracy, best streak, and your worst /
  best-known words.

## Develop

```bash
bun install
bun run dev      # http://localhost:5173
```

Other scripts:

```bash
bun run build      # production build → dist/
bun run preview    # preview the build locally
bun run typecheck  # tsc, no emit
bun run lint       # oxlint
```

## Data

Word pairs come from `src/data/top_1000_words.csv` (columns: `Dutch,English`).
It's imported with Vite's `?raw` suffix and parsed at build time — no runtime
fetch, no backend.

## Scores

Scores are stored in `localStorage` under `nl-cards:scores:v1` (per word) and
`nl-cards:session:v1` (current session streak/accuracy). This keeps the app
fully static and free to host. To switch to server-backed persistence later
(e.g. cross-device sync), replace the store in `src/lib/scores.ts` — the
`useScores` API (`getScore`, `recordResult`, `resetAll`, `scores`, `session`)
won't need to change.
